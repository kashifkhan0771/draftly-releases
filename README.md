# Draftly

A local-first, privacy-native note-taking app for your desktop.

→ **[Download the latest release](https://github.com/kashifkhan0771/draftly-releases/releases/latest)**

---

## Install

### Linux (Ubuntu / Debian / Mint)

```bash
sudo apt install ./draftly_VERSION_amd64.deb
```

### macOS

| Your Mac | File |
|----------|------|
| Apple Silicon (M1/M2/M3) | `draftly-macos-arm64` |
| Intel | `draftly-macos-amd64` |

```bash
chmod +x draftly-macos-*
xattr -dr com.apple.quarantine draftly-macos-*
./draftly-macos-*
```

### Windows

Run `draftly-windows-amd64.exe` and follow the installer.

---

## Update

**Linux** — download the new `.deb` and run the same install command:
```bash
sudo apt install ./draftly_NEW_VERSION_amd64.deb
```

**macOS** — replace the existing binary with the new download.

**Windows** — run the new installer; it replaces the old version automatically.
